# lab-main-index

本仓库用于归档和发布多个独立 HTML 页面。页面内容涵盖技术示例、实验性原型、工具展示等，不针对任何单一网站或具体域名。

## 项目简介

本项目是一个 HTML 页面集合的归档仓库。页面均以静态 HTML 文件形式存放，可直接通过浏览器访问。仓库主要用于保存和索引各类独立页面，便于整理、回顾和共享。

## 目录说明

```
lab-main-index/
├── index.html          # 仓库主页，列出所有归档页面链接
├── pages/              # 存放各个独立 HTML 页面
│   ├── page-a.html
│   ├── page-b.html
│   └── ...
└── assets/             # 页面所需的资源文件（如图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：仓库入口页面，提供所有归档页面的导航。
- `pages/`：每个 HTML 页面独立存放，命名清晰，便于识别。
- `assets/`：公共资源目录，各页面可引用其中的文件。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，不依赖外部数据库或后端服务。
- 页面内容可能包含实验性功能或临时示例，不保证长期可用性。
- 页面命名建议使用有意义的英文或拼音，避免特殊字符。
- 新增页面时，请同时在 `index.html` 中添加对应链接。
